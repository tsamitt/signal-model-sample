# Signal Model Sample

This is a small Python sample showing a basic time-ordered machine learning workflow for structured signal data.

## What it does
- Cleans missing values
- Splits data chronologically to avoid look-ahead bias
- Trains an XGBoost classifier
- Prints classification metrics

## Why I made it
I created this as a simple example of how I approach feature-based classification problems with time-ordered data.

## Tech used
- Python
- pandas
- scikit-learn
- xgboost
